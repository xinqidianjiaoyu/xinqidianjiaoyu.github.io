hexo搭建博客

#hexo

##hexo搭建本地博客

  1. 全局安装hexo npm install hexo -g
  2. 创建一个blog文件夹,在blog文件夹中执行 hexo init 初始化hexo , 会自动下载hexo的依赖
  3. 执行命令生成静态html文件 hexo generate
  4. 执行 hexo server 会生成一个服务器 并监听4000端口 输入 http://localhost:4000 即可访问本地博客

配置Guthub

  1. 建立Repository 名称必须为 xinqidianjiaoyu(userName).github.io

  2. 然后打开blog文件夹下面的 _config.yml 文件 翻到最下面, 配置deploy

    deploy:

     type: git

     repo: https://github.com/leopardpan/leopardpan.github.io.git

     branch: master

  3. 安装 hexo-deployer-git 插件(让文件支持git上传)

  4. 最后执行命令 hexo deploy 把本地的public中的静态博客部署到github的仓库中
  5. 在浏览器中输入 https://xinqidianjiaoyu.github.io 就可访问github上的博客了

###每次使用指令步骤

每次部署的步骤，可按以下三步来进行。

hexo clean    清空public文件夹

hexo generate    根据 source/_posts/*.md文件生成public文件夹和对应的网页文件

hexo deploy  把本地public文件夹中的网站资源上传到github服务器中

通过 https://xinqidianjiaoyu.github.io 就可以访问网站博客了

一些常用命令：

hexo new"postName" #新建文章

hexo new page"pageName" #新建页面

hexo generate #生成静态页面至public目录

hexo server #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）

hexo deploy #将.deploy目录部署到GitHub

hexo help # 查看帮助

hexo version #查看Hexo的版本
