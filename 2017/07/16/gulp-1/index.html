gulp

#gulp

  1. 在gulp跟文件中使用 npm init 创建 package.json文件
  2. 使用 npm install gulp –save-dev 添加项目在开发阶段对gulp的依赖并下载gulp,因为gulp只是在开发阶段使用,不在项目运行阶段使用,所有只需要在开发依赖中使用即可
  3. 在项目跟目录中创建gulpfile.js文件去设置需要用到的任务,文件名是固定的不可以自定义文件名
  4. 编译gulpfile设置各种任务

  5. gulp执行各项任务需要依赖的包

    • gulp-less 编译less的工具
    • gulp-concat 合并代码
    • gulp-uglify 压缩js代码
    • gulp-cssnano 压缩css
    • gulp-htmlmin 压缩html
    • browser-sync 自动化同步服务器

  6. 编译gulpfile,把相应的任务写好以后,在命令行中输入 gulp style(任务名称)去执行相应任务
