call()

call(obj, arg1, arg2);
  1. 把调用call()函数的方法内的this指针指向call()里的第一个参数
  2. 也就是说调用call()的函数的实现给了call()的第一个参数

  3. call()里的第一个参数是调用call()的对象,其余参数也要是对象,从第二个参数开始后的参数会当成参数传进call()的方法调用里

    (function () {
        var obj = {
        name: "jack"
    };
    obj.showM = function () {
        console.log(this, arguments[0].name, arguments[1].age);
        //Object {name: "kai"} "kai" 20
    };

    var obj1 = {
           name: "kai"
    };
    var obj2 = {
    age: 20
    };

    obj.showM.call(obj1, obj1, obj2); 
})()

  4. 在这个方法中把obj.showM的方法实现,交给了obj1, 并且分别把obj1,和obj2当做参数传入了obj1调用obj.showM方法的过程中

apply()

apply(obj, [arg1, arg2])
  1. apply()的用法跟call()基本一致,只是语法稍有不同
  2. call()中的参数都为对象,且从第二个对象开始就是会当做参数传递
  3. apply()中只有两个参数,第一个参数为调用方法的对象,第二个参数是一个数组,执行时会把数组中的每个元素当做参数传递到方法的调用过程中
function bb() {
    console.log()this, arguments); 
    //{name: "jackChen, age: 24}   [hao a zhang kai](伪数组 有 0,1,2,3,callee,length属性)
}

var kai = {
    name: "jackChen",
    age: 24
};

var obj = {
    name: "jssts"
}

obj.cc = function () {
    return bb.apply(this, Array.prototype.slice.call(arguments, 1));
}

obj.cc.call(kai, "ni","hao","a","zhang","kai");

  • 在此案例中首先要明白Array.prototype,slice(start,end)方法

    • 这个方法本身为数组的截取方法,是从start角标到end角标,包含start不包含end [start end),而且返回的是一个截取好的数组
    • 如果不传end默认是从start开始截取到数组末尾
    • 而通过调用Array.prototype.slice.call(arguments, 1)可以把是伪数组的auguments弄成一个数组返回,而后边的1则是从角标1截取到数组末尾
    • arguments 中的length属性决定数组的长短,而且auguments中的属性必须是0–> 根据length去生成原有的角标的数组

  • 根据 Array.prototype.slice.call(arguments, 1) 方法返回的是[“hao”, “a”, “zhang”, “kai”]

  • 此时obj.cc方法中的this是kai这个对象
  • obj.cc中的方法中的返回值是执行bb方法,bb.apply()中的this是kai这个对象,第二个参数的数组中的每一项会当成参数传进bb的方法中执行
  • 也就是说最后的结果就是kai这个对象在调用bb的方法实现,并且带有参数,所以打印的结果是this是kai这个对象,arguments是一个伪数组
