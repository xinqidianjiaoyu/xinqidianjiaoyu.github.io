npm&bower

npm&bower

  • npm是一个包管理工具(node package manager)
  • 在项目跟文件夹下,使用命令npm install jquery 可以安装包,安装后会在node_modules中把指定的包下载到里面,并自动把其依赖的包也会下载
  • 在install之前使用命令npm init 创建一个package.json文件然后在下载包的时候使用命令npm install jquery –save 会在package.json中有一个dependencies节点记录本项目依赖的包,在下次使用的时候可以直接使用npm install命令去下载项目依赖的包.
  • 使用npm inistall jquery –save-dev会在package.json中有一个devDependencies节点记录项目开发阶段使用到的包,而跟项目运行没有关系.
  • 使用npm install jquery -g 去下载全局用到的包, 比如bower
  • 使用npm uninstall jquery 去卸载一个包
  • bower跟npm区别
    • bower主要用于Web页面开发时使用的包.
    • npm主要用于node开发时使用的包.
